Brief 21 — KSeF (Krajowy System e-Faktur) — wersja rozszerzona

Kontekst
Dlaczego KSeF jest krytyczny
Od 1 lutego 2026 firmy z przychodem >200M PLN MUSZĄ wysyłać faktury przez KSeF. Od 1 kwietnia 2026 wszystkie pozostałe firmy. To nie jest opcja — to wymóg prawny. Faktura wystawiona poza KSeF nie będzie uznawana.
Dla AutoFlow to ogromna szansa — firmy masowo szukają narzędzi do integracji z KSeF. AutoFlow z wbudowanym KSeF = przewaga nad Zapier/Make które tego nie mają.
Dla FastHub to fundament — każdy polski SaaS z billingiem będzie potrzebował KSeF. FastHub z gotowym klientem KSeF = oszczędność tygodni pracy dla developerów.
Co mamy
Provider KSeF dla AutoFlow (680 linii) — pełna integracja:

Sesje interaktywne i wsadowe (batch)
Wysyłanie faktur FA(3) XML
Walidacja faktur
Generowanie XML (helper)
Odbieranie faktur (sprzedaż + koszty)
UPO (Urzędowe Poświadczenie Odbioru)
Uprawnienia i certyfikaty
Auth v2.0 (challenge → JWT)
Status systemu (latarnia)
Wizualizacja faktur (HTML/PDF)

Co robimy
Wyciągamy warstwę HTTP (klient) do fasthub_core/clients/ksef_client.py. Provider w AutoFlow potem importuje tego klienta zamiast mieć własną logikę HTTP. Dokładnie wzorzec Shared HTTP Client z Brief 16:
fasthub_core/clients/ksef_client.py    ← warstwa HTTP (auth, sesje, API calls)
        ↑                                      ↑
AutoFlow Provider                    FastHub billing hook
(klient automatyzuje                 (MY wystawiamy faktury
 SWOJE faktury)                       klientom przez KSeF)
Plus hook do auto-fakturowania — alternatywa dla Fakturowni. Po płatności → zamiast Fakturownia → KSeF. Konfigurowalne — co kto woli.

Dual context — jak to wygląda w praktyce
Kontekst 1 — Provider (AutoFlow):
Pani Zosia z firmy XYZ używa AutoFlow. Podłącza swój NIP i token KSeF. Buduje workflow: "gdy zamówienie w Allegro → wystaw fakturę przez KSeF → wyślij PDF klientowi". KSeFClient używa tokenu pani Zosi.
Kontekst 2 — Billing (FastHub/AutoFlow jako firma):
AutoFlow Sp. z o.o. sprzedaje subskrypcję panu Kowalskiemu. Po płatności hook automatycznie wystawia fakturę przez KSeF używając NIPu i tokenu AutoFlow. KSeFClient.from_config() czyta z .env.

"""
KSeF Client — HTTP client do Krajowego Systemu e-Faktur.

Dwa tryby:
    1. Provider: KSeFClient(nip="1234567890", auth_token="xxx", environment="prod")
    2. Billing:  KSeFClient.from_config()  → czyta KSEF_* z .env

Docs API: https://ksef.mf.gov.pl/api/v2
Środowiska:
    - prod: https://ksef.mf.gov.pl/api/v2
    - demo: https://ksef-demo.mf.gov.pl/api/v2 
    - test: https://ksef-test.mf.gov.pl/api/v2
"""

class KSeFClient(BaseHTTPClient):
    
    ENV_URLS = {
        "prod": "https://ksef.mf.gov.pl/api/v2",
        "demo": "https://ksef-demo.mf.gov.pl/api/v2",
        "test": "https://ksef-test.mf.gov.pl/api/v2",
    }
    LATARNIA_URL = "https://api-latarnia.ksef.mf.gov.pl"

    def __init__(self, nip: str, auth_method: str = "token",
                 auth_token: str = "", certificate_base64: str = "",
                 private_key_base64: str = "", environment: str = "test"):
        base_url = self.ENV_URLS.get(environment, self.ENV_URLS["test"])
        super().__init__(base_url=base_url)
        self.nip = nip.replace("-", "").replace(" ", "")
        self.auth_method = auth_method
        self.auth_token = auth_token
        self.certificate_base64 = certificate_base64
        self.private_key_base64 = private_key_base64
        self.environment = environment
        self._access_token: Optional[str] = None

    @classmethod
    def from_config(cls) -> "KSeFClient":
        """
        Billing mode — czyta config z .env.
        
        Wymagane:
            KSEF_NIP=1234567890
            KSEF_AUTH_TOKEN=xxx
            KSEF_ENVIRONMENT=prod|demo|test
        """
        from fasthub_core.config import get_settings
        settings = get_settings()
        return cls(
            nip=settings.KSEF_NIP,
            auth_method=getattr(settings, "KSEF_AUTH_METHOD", "token"),
            auth_token=settings.KSEF_AUTH_TOKEN,
            certificate_base64=getattr(settings, "KSEF_CERTIFICATE_BASE64", ""),
            private_key_base64=getattr(settings, "KSEF_PRIVATE_KEY_BASE64", ""),
            environment=getattr(settings, "KSEF_ENVIRONMENT", "test"),
        )

    def _session_headers(self) -> Dict[str, str]:
        headers = {"Content-Type": "application/json", "Accept": "application/json"}
        if self._access_token:
            headers["Authorization"] = f"Bearer {self._access_token}"
        return headers

"""
Generator faktur XML zgodnych ze schematem FA(3).

Buduje XML lokalnie — nie wymaga sesji KSeF.
Używany zarówno przez Provider (AutoFlow) jak i billing hook.

class KSeFXMLBuilder:
    """
    Builder faktur XML FA(3) dla KSeF.
    
    Użycie:
        builder = KSeFXMLBuilder()
        result = builder.build(
            seller_nip="1234567890",
            seller_name="AutoFlow Sp. z o.o.",
            buyer_nip="9876543210",
            buyer_name="Firma Klienta",
            issue_date="2026-03-02",
            sale_date="2026-03-02",
            positions=[
                {"name": "Plan Pro — miesięczny", "quantity": 1, "unit": "szt",
                 "price_net": 199.00, "vat_rate": 23},
            ],
            currency="PLN",
            payment_method="przelew",
            payment_deadline="2026-03-16",
            bank_account="PL12345678901234567890123456",
            system_info="AutoFlow",
        )
        
        result["invoice_xml"]         — XML string
        result["invoice_xml_base64"]  — base64 do wysyłki
        result["summary"]            — podsumowanie (netto, VAT, brutto)
    """
    
    def build(self, seller_nip, seller_name, buyer_nip, buyer_name,
              issue_date, sale_date, positions, currency="PLN",
              payment_method=None, payment_deadline=None,
              bank_account=None, notes=None, system_info="FastHub") -> Dict:
        """Generuj XML FA(3). Logika z providera AutoFlow."""
        ...  # logika z generate_invoice_xml providera

    def validate_nip(self, nip: str) -> bool:
        """Walidacja NIP (cyfra kontrolna)."""
        nip = nip.replace("-", "").replace(" ", "")
        if len(nip) != 10 or not nip.isdigit():
            return False
        weights = [6, 5, 7, 2, 3, 4, 5, 6, 7]
        checksum = sum(int(nip[i]) * weights[i] for i in range(9)) % 11
        return checksum == int(nip[9])

    def validate_positions(self, positions: List[Dict]) -> List[str]:
        """Walidacja pozycji faktury. Zwróć listę błędów (pusta = OK)."""
        errors = []
        for i, pos in enumerate(positions):
            if not pos.get("name"):
                errors.append(f"Pozycja {i+1}: brak nazwy")
            if pos.get("price_net", 0) <= 0:
                errors.append(f"Pozycja {i+1}: cena netto <= 0")
            if pos.get("quantity", 0) <= 0:
                errors.append(f"Pozycja {i+1}: ilość <= 0")
            if pos.get("vat_rate") not in (0, 5, 8, 23, -1):  # -1 = zw
                errors.append(f"Pozycja {i+1}: nieprawidłowa stawka VAT")
        return errors

KSeFInvoiceHook — auto-faktura przez KSeF
Alternatywa do AutoInvoiceHook (Fakturownia) z Brief 19. Wybór: Fakturownia LUB KSeF — konfigurowalny.
python# fasthub_core/billing/ksef_hook.py

"""
Auto-faktura przez KSeF po udanej płatności.

Alternatywa do AutoInvoiceHook (Fakturownia).
Config: INVOICE_BACKEND=ksef|fakturownia|none

Flow:
    1. Płatność przechodzi (webhook)
    2. Hook pobiera dane organizacji (NIP, adres)
    3. KSeFXMLBuilder generuje XML FA(3)
    4. KSeFClient otwiera sesję → wysyła fakturę → zamyka sesję
    5. Numer KSeF zapisany w BillingEvent
    6. UPO pobrane i zapisane
"""

class KSeFInvoiceHook:
    """
    Hook: payment_succeeded → wystaw fakturę przez KSeF.
    
    Użycie:
        hook = KSeFInvoiceHook(db)
        stripe_handler.on_payment_succeeded = hook.handle
    
    Wymaga w .env:
        KSEF_NIP, KSEF_AUTH_TOKEN, KSEF_ENVIRONMENT
        INVOICE_SELLER_NAME, INVOICE_SELLER_ADDRESS (dane na fakturze)
    """
    
    async def handle(self, event_data: Dict):
        """
        1. Pobierz dane kupującego (org: NIP, nazwa, adres)
        2. Zbuduj XML (KSeFXMLBuilder)
        3. Waliduj XML (KSeFClient.validate_invoice)
        4. Otwórz sesję (KSeFClient.open_session)
        5. Wyślij fakturę (KSeFClient.send_invoice)
        6. Zamknij sesję (KSeFClient.close_session)
        7. Pobierz UPO (KSeFClient.get_upo)
        8. Zapisz numer KSeF w BillingEvent
        """
        tenant_id = event_data.get("tenant_id")
        if not tenant_id:
            return
        
        org = await self._get_organization(tenant_id)
        if not org or not org.nip:
            logger.warning(f"KSeF: org {tenant_id} has no NIP, skipping")
            return
        
        # Build XML
        builder = KSeFXMLBuilder()
        xml_result = builder.build(
            seller_nip=settings.KSEF_NIP,
            seller_name=settings.INVOICE_SELLER_NAME,
            buyer_nip=org.nip,
            buyer_name=org.name,
            issue_date=date.today().isoformat(),
            sale_date=date.today().isoformat(),
            positions=[{
                "name": event_data.get("plan_name", "Subskrypcja"),
                "quantity": 1,
                "unit": "szt",
                "price_net": event_data.get("amount", 0) / 100 / 1.23,  # brutto → netto
                "vat_rate": 23,
            }],
            payment_method="przelew",
            bank_account=settings.INVOICE_BANK_ACCOUNT,
            system_info=settings.APP_NAME,
        )
        
        # Send to KSeF
        client = KSeFClient.from_config()
        
        # Walidacja
        validation = await client.validate_invoice(xml_result["invoice_xml_base64"])
        if not validation.get("valid", True):
            logger.error(f"KSeF validation failed: {validation}")
            return
        
        # Sesja → wysyłka → zamknięcie
        session = await client.open_session()
        reference = session.get("referenceNumber")
        
        try:
            invoice_result = await client.send_invoice(reference, xml_result["invoice_xml_base64"])
            ksef_number = invoice_result.get("elementReferenceNumber")
        finally:
            await client.close_session(reference)
        
        # UPO
        upo = await client.get_upo(reference)
        
        return {
            "ksef_number": ksef_number,
            "reference_number": reference,
            "upo": upo,
            "xml_summary": xml_result["summary"],
        }

def get_invoice_hook(db):
    """Factory — zwróć hook na podstawie config."""
    backend = settings.INVOICE_BACKEND
    
    if backend == "ksef":
        from fasthub_core.billing.ksef_hook import KSeFInvoiceHook
        return KSeFInvoiceHook(db)
    elif backend == "fakturownia":
        from fasthub_core.billing.invoice_hook import AutoInvoiceHook
        return AutoInvoiceHook(db)
    else:
        return None  # brak auto-faktur
